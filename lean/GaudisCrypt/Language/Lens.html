                                                      Navigating nested tuples #

                                                      A TuplePath names a slot in an arbitrarily nested tuple of binary products (here/left/right); the ProjAt typeclass decomposes the (concrete) tuple type structurally, so Lens.insideTuple path deduces both the tuple type M and the component type A from the expected output type. Encoding the path as an inductive (rather than Nat arithmetic in instance indices) keeps typeclass resolution robust.
