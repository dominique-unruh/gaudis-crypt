                      structure GaudisCrypt.Language.Programs.LocalVariableState (paramTypes : List Type) (locals : List ((t : Type) × Inhabited t)) :
                      Type

                      The local state of a procedure: parameter values (params) and local-variable values (vars). Indexed by the parameter types and the local declarations only (not the return type), so it can be formed before the return type is known — this is what lets a proc with an omitted return type elaborate.

                      • params : paramListToTuple paramTypes
                      • vars : paramListToTuple (List.map (fun (x : (t : Type) × Inhabited t) => x.fst) locals)
