              def LensRange.global_getter {m : Type u_1} (R : LensRange m) :
              GaudisCrypt.Language.Lens.Getter (Quotient R.orbit_setoid) m

              The "global getter" of a LensRange: the quotient projection onto orbit-classes.

              Reading: two states give the same getter value iff they are in the same R-orbit.

              For a lens-derived range R = l.range, two states are in the same R-orbit iff they differ only in l's content — so this getter encodes the complement of l.

              Convention: glob A is typically "what A touches", i.e. the commutant's orbits, so one writes glob A := A.range.commutant.global_getter (commutant = Rᶜ). Equivalently glob A := A.rangeᶜ.global_getter.

              Equations
              • R.global_getter = { get := Quotient.mk R.orbit_setoid }
